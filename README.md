Soy desarrollador backend en formación, cursando la Licenciatura en Análisis de Sistemas en la UTIC y el programa intensivo CodePRO de Penguin Academy. Me oriento hacia el desarrollo web full-stack, con bases sólidas en backend y con interés creciente en el lado del frontend.
